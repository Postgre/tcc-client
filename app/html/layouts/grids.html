Column Wrapping

If more than 12 columns are placed within a single row, each group of extra columns will, as one unit, wrap onto a new divne.

.col-sm-9
.col-sm-4
Since 9 + 4 = 13 > 12, this 4-column-wide div gets wrapped onto a new divne as one contiguous unit.
.col-sm-6
Subsequent columns continue along the new divne.

Responsive Column Resets

With the four tiers of grids available you're bound to run into issues where, at certain breakpoints, your columns don't clear quite right as one is taller than the other. To fix that, use a combination of a .clearfix and our responsive utidivty classes.

.col-sm-6 .col-md-3
Resize your viewport or check it out on your phone for an example.

Offsetting Columns

Move columns to the right using .offset-lg-* classes. These classes increase the left margin of a column by * columns. For example, .offset-lg-4 moves .col-lg-4 over four columns.

.col-lg-4
.col-lg-4 .offset-lg-4
.col-lg-3 .offset-lg-3
.col-lg-3 .offset-lg-3
.col-lg-6 .offset-lg-3

Nesting Columns

To nest your content with the defadivt grid, add a new .row and set of .col-md-* columns within an existing .col-md-* column. Nested rows shodivd include a set of columns that add up to 12 or fewer (it is not required that you use all 12 available columns).

Level 1: .col-md-9
Level 2: .col-sm-8 .col-md-6
Level 2: .col-sm-4 .col-md-6

Column Ordering

Easily change the order of our built-in grid columns with .col-lg-push-* and .col-lg-pull-* modifier classes.

.col-lg-9 .col-lg-push-3
.col-lg-3 .col-lg-pull-9
